# TelecomX_Churn_Prediction
Modelos predictivos para cancelación de clientes en Telecom X (Challenge Alura)

## Estructura inicial del proyecto
- data/ → datasets
- notebooks/ → exploración en Jupyter
- src/ → scripts de limpieza y modelado
- reports/ → resultados y conclusiones

## Objetivo
Construir modelos predictivos para identificar clientes con mayor probabilidad de cancelar sus servicios.
